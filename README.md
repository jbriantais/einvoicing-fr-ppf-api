# einvoicing-fr-ppf-api
REST API documentation of e-invoicing public portal (PPF) for future CTC requirements in France

## About this repository
This repository contains tools for extracting OpenAPI specifications from the [official specifications of the PPF](https://www.impots.gouv.fr/specifications-externes-b2b) and deploying a Swagger UI documentation site to GitHub pages.
